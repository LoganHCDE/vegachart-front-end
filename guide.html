Chart Guide

Chart Types

Descriptions for chart types and their use cases.

Scatter Plot

Shows relationship between two numeric variables. Look for clusters, outliers, and trends.

Line Plot

Continuous line across ordered x values. Use for progression or change over time.

Bar Plot

Compare categories with heights. Great for counts and aggregated values.

Box Plot

Summarizes distributions by category: median, quartiles, and outliers.

Violin Plot

Like a box plot but shows density across values for richer comparison.

Dot Plot

Displays individual observations or category values as points for simple comparison.

Pie Chart

Shows part-to-whole composition. Use sparingly and with few slices.

Jitter Plot

Adds random noise to reduce overlap when many points share similar positions.

Bar Chart

Bar chart of category frequencies. Ideal for quick distributions of labels.

Best practice #1

Use clear titles, axis labels, and source text. Keep color use purposeful and consistent with a palette that supports accessibility. Avoid clutter by removing non-essential gridlines, borders, and decorations.

Illustration

Best practice #2

Match the chart type to the question: bars for comparing categories, lines for change over ordered values, scatter for relationships. Summarize distributions with box or violin plots when you need to compare spread.

Illustration

Best practice #3

Annotate important values and use legends sparingly. When possible, label directly. Always include units, and ensure text contrast is high enough for readability on dark backgrounds.
